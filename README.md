# vue 渐进式框架

## 了解 vue

Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。完全能够为复杂的单页应用提供驱动，典型的 MVVM 框架，即开发中只需要注重逻辑，不需要关心视图。
